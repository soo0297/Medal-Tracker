# Olympic Medal Tracker 만들기 
## : React 입문주차 개인과제
### ⚙ features
- 제출 폼 UI 구현하기
- 메달 집계 CRUD 구현하기
    - Create: 새로운 나라와 그 나라가 획득한 메달 수를 추가
    - Read: 나라별 메달 집계 리스트를 보여주기
    - Update: 기존에 추가된 나라의 메달 수를 수정
    - Delete: 나라 정보를 삭제
- 정렬 (구현 못함.)
- 컴포넌트 분리는 하지 않았습니다.
